# レンズ何持ってく？

https://lensdore-c55ce.web.app/

カメラマン向けの Web アプリ

その名の通り、レンズの焦点距離と各種撮影条件を指定すると、
撮影画像のシミュレーションが表示できます。
撮影時のレンズ・機材選びのお供にどうぞ。
